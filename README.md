# PFVMUSIC report5

Проект подготовлен для локального запуска как статический сайт.

## Запуск

```bash
npm start
```

По умолчанию сервер поднимается на `http://127.0.0.1:3000` и открывает `report5.html` через `index.html`.

## Полезно

- Основной файл отчёта: `report5.html`
- Точка входа: `index.html`
- Локальный сервер: `server.js`

## Примечание

Внутри `report5.html` используются внешние ресурсы:

- `cdnjs` для библиотеки `xlsx`
- `fonts.googleapis.com` для шрифтов

Поэтому для полного отображения интерфейса и Excel-экспорта в браузере нужен доступ в интернет.
